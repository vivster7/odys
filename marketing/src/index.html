When should I use this?

If you're collaborating remotely and find yourself reaching for a whiteboard, this tool is perfect. Common usecases include onboarding a new hire, conducting a remote interview, or giving a brownbag talk.

?

Is this a documentation tool?

You're welcome to save a screenshot, but there are better tools if you're just looking to make a static image to save as canonical documentation.

?

Is this a presentation tool?

It's perfect for an ad-hoc presentation to share knowledge with your team. But if you're preparing a presentation for a large audience, you might want to stick to Powerpoint (unless you're David Beazley) .
